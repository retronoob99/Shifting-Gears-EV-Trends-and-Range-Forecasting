# üìå Project Overview

This project explores the shift toward sustainable transportation by analyzing the adoption trends of Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs) across Washington State. Using real-world electric vehicle registration data, the analysis uncovers geographic, manufacturer, and performance-based insights, along with a predictive machine learning model.

The goal of this project is to understand market trends, policy impact, and key drivers behind EV adoption while demonstrating end-to-end data analysis and modeling skills.

Link of the dataset --> https://catalog.data.gov/dataset/electric-vehicle-population-data

# üîç Key Objectives

Analyze BEV vs. PHEV adoption trends across counties and cities

Identify factors influencing electric vehicle eligibility and adoption

Build a machine learning model to predict vehicle type

Visualize insights to support data-driven sustainability discussions

# üßπ Data Cleaning & Preprocessing

Cleaned and preprocessed EV registration data using Pandas

Handled missing values and standardized categorical fields

Engineered features such as electric range impact on CAFV eligibility

Ensured data consistency for accurate analysis and modeling

# üìä Exploratory Data Analysis & Visualization

Visual analysis was conducted using Matplotlib and Seaborn to uncover:

Growing dominance of BEVs over PHEVs

Adoption patterns across counties, cities, and manufacturers

Geographic concentration of EV adoption aligned with state sustainability initiatives

Relationship between electric range and policy eligibility

Achieved ~99% model accuracy, indicating strong feature relevance and data quality

# üí° Key Insights

BEVs are increasingly preferred over PHEVs across Washington State

This project provides a comprehensive view of the evolving electric vehicle landscape in Washington State, highlighting a clear shift toward Battery Electric Vehicles as the preferred sustainable transportation choice.
